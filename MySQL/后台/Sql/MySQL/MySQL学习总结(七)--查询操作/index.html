MySQL学习总结(七)--查询操作

MySQL MySQL 2018/06/26

数据库最重要的一个功能就是数据查询，数据查询不仅需要对数据进行筛选，而且要确定数据以什么格式显示，今天我们就来了解下查询操作。

基本查询语句

语法:

  • SELECT：表示查询操作
  • 字段列表：表示查询的字段，多个字段间以逗号间隔。
  • FROM 表名1，表名2……：数据的来源，可以是单个或多个。
  • WHERE：可选项，表示数据筛选的条件
  • GROUP BY 分组字段 HAVING表达式: GROUP BY 按照指定的字段进行分组
  • ORDEER BY 排序字段 [ASC | DESC]：将数据根据指定的字段进行排序，默认为升序排序(ASC)。
  • LIMIT [offset，] [rows]：需要显示的条数

单表查询

查询所有字段

可以通过 * 通配符的形式查询，当然也可以通过指定字段查询（推荐使用此方式）。

通过指定字段方式，可以只显示需要的字段名，而不用全部显示所有字段。

查询指定记录

数据库中存在大量的数据，根据需求可能只需要表中部分数据，这个时候可以通过 WHERE 对数据进行筛选。

BETWEEN AND 在范围内的数据
LIKE 和通配符结合模糊查询数据
IS NULL 查询 NULL 值的数据
AND 多个条件都符合的数据
