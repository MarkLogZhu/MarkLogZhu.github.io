IDEA 创建 JAVA 项目

• 2017年11月13日

1) 运行 IDEA

2) 对话框中依次点击

2.1) 选中 Java Enterprise 选项卡
2.2) 勾选 Web Application(3.1) 复选框，这表示我们新建的项目是个网站。
2.3) 点击 Next 按钮，这会在 project下添加一个默认的与 project 同名的 module。
