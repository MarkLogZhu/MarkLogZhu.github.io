SSM框架统一异常处理

java Java SSM 2018/05/09

在企业级项目使用中出现异常是很正常的一件事情，但是如果直接将异常信息给用户看到是一件很不好的事情，而且用户反馈的时候也很有可能说不清异常问题，所以通过统一异常处理将错误信息处理成用户可以理解的方式是很有必要的！ 这篇文章是在上一篇 封装返回结果JSON格式 的基础上增加的，如果没有了解的同学可以先去看下！
