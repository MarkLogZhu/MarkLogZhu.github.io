JavaScript 之 DOM 的使用

前端基础 CSS DOM 2017/09/16

一、什么是 DOM

DOM 全称是 Document Object Model，也就是文档对象模型。

dom 是一个文档对象，他将html文件按节点生成一个文档对象

二、它能干什么

它可以让我们操作对象一样的操作文档，来进行增删改操作。

三、节点

DOM 有节点组成，是同一个集合对象，它有如下三种节点：

  • 元素节点
  • 文本节点
  • 属性节点

3.1 元素节点

HTML 中的标签就是元素节点,如 <a>、<p> 标签等。

3.2 文本节点

文本节点是指如被 <a> 标签包含的文字内容，元素不一定都包含文本节点，如<ul>。

3.3 属性节点

属性节点上被元素节点包含的，如 title 属性。

四、节点属性

4.1 childNodes 属性

通过 childNodes 属性可以获取一个元素下所包含的所有子元素，返回的是一个数组对象。

var body = document.getElementsByTagName("body")[0];
var array = body.childNodes;

4.2 nodeType 属性

每个节点都有 nodeType 属性，可以让我们区分节点类型。它返回的值是数字，含义如下：

* 元素节点：1
* 属性节点：2
* 文本节点：3

4.4 firstChild 属性

first 英文翻译过来就是第一

我们知道 childNodes 属性返回的是一个数组对象，如果要获取他的第一个对象的话，除了根据数组下标来取之外，也可以通过 firstChild 属性来获取。

4.5 lastChild 属性

与 firstChild 属性相对应的就是 lastChild 属性,它表示最后一个对象。

五、设置/获取 元素节点属性

5.1 获取属性

// 获取属性值

5.2 设置属性

// 设置属性值
setAttribute("属性名称",值);

replaceData(offset,length,string) 方法用于替换文本节点中的数据。

offset - 在何处开始替换字符。Offset 值以 0 开始。
length - 要替换多少字符

nodeValue 属性 也可以实现完全替换文本数据。

6.6.1 追加节点内容

appendChild() 方法向已存在的节点添加子节点，新节点会添加（追加）到任何已存在的子节点之后。

6.6.2 插入节点

insertBefore() 方法用于在指定的子节点之前插入节点。
