Velocity实战入门(二)--取值

Velocity Velocity 2018/04/25

取值方式

在上一篇中我们已经搭建了一个入门案例，我们可以看到在后台给前台返回了一个属性 

@RequestMapping("/index")
public String vmIndex(Model model) {
    model.addAttribute("hello", "hello velocity");
    return "test";
}

那么 Velocity 是怎么取值的呢？

我们来看下上个案例中的 test.vm 页面

<html>
<head>
    <title>Spring MVC and Velocity</title>
</head>
<body>
    $hello
</body>
</html>

可以发现 它是通过 $ + 变量名 的形式来换取值的

我们现在只是赋值了一个字符串，那么我们试试看能不能传递下对象

示例

1) 新建 pojo 包并在下面新建一个 User.java 

public class User {

    private String userName;
    private int age;

    public String print(String userName,int age){
        return "我叫"+userName+"今年"+age + "岁了";
    }
    @Override
    public String toString() {
        return "User{" +
            "姓名='" + userName + '\'' +
            ", 年龄=" + age +
            '}';
    }
    // get set ...
}

2) 在 HelloController.java 下新增方法

@RequestMapping("/index2")
public String vmIndex2(Model model) {
    model.addAttribute("user", initUser());
    model.addAttribute("args",initArgs());
    return "test2";
}

private User initUser(){
    User user = new User();
    user.setUserName("marklogzhu");
    user.setAge(18);
    return user;
}

private String[] initArgs(){
    return new String[]{"张三","李四","王五","赵六"};
}

3) 在 templates 目类下新建 test2.vm 文件

<html>
<head>
    <title>Spring MVC and Velocity</title>
</head>
<body>
    <h3>对象的使用</h3>
    user 对象: $user
    <h3>方法的使用</h3>
    user.toString(): $user.toString()
    <br>
    user.getUserName() : $user.getUserName()
    <br>
    user.getAge() : $user.getAge()
    <br>
    user .print(): $user.print($user.userName,$user.age)
    <br>
    <h3>属性的使用</h3>
    user.age 属性 : $user.age
    <br>
    user.userName 属性 : $user.userName
    <h3>不存在的对象的使用</h3>
    admin字符串 ： $admin
    <br>
    admin字符串 ：$!{admin}
    <h3>数组索引的使用</h3>
    args : $!{args[0]},$!{args[1]},$!{args[2]}
</body>
</html>

4) 启动项目,在浏览器中输入 http://localhost:8080/hello/index2

从上面的结果我们可以发现：

* 所有的引用的值都被转为了 String 类型, 就算是对象也会被调用 toString() 方法
* 可以直接引用对象的属性
* 可以直接调用对象的方法并且可以传参    
* $admin 这种写法如果没有这个值的话就会原样输出，而通过 $!{admin} 就不会显示在页面中了
* ${admin} 的写法不会在与其他文本相关联的时候 让 Velocity 解析错误如： $adminUsername
* 数组也可以通过索引的形式输出

引用

取值的学名叫做引用，

正规的引用格式

语法:

${mudSlinger}
${customer.Address}
${purchase.getTotal()}

简写的引用格式

语法:

$mudSlinger
$customer.Address
$purchase.getTotal()

值为null 时不输出的引用格式

语法:

$!{admin}
