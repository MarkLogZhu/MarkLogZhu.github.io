执行后会返回如下一张表结构：

字段含义如下：

  • NULL:表示该列算法可以存储 NULL 值
  • Key:表示该列是否已编制索引。
    • PRI:表示该列是表主键的一部分
    • UNI:表示该列是UNIOUE索引的一部分
    • MUL:表示在列中某个给定值允许出现多次
  • Default:表示该列是否有默认值，如果有的话值是多次
  • Extra:表示可以获取的与给定列有关的附加信息，如 AUTO_INCREMENT等。
