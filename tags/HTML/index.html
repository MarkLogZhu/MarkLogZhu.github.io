一、HTML概念描述

1.什么是 HTML

HTML 是英文 Hyper Text Mark-up Language( 超文本标记语言 )的缩写，它规定了自己的语法规则，用来表示比“文本”更丰富的意义，比如图片，表格，链接等。

简单来说 HTML 就是给文本加上各种表明文本含义的标签，让人和程序对文本有更直观的认识。
