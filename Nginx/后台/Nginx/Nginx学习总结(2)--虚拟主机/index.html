Nginx学习总结(2)--虚拟主机"

Nginx Nginx 2018/04/13

什么是虚拟主机

虚拟主机是指在网络服务器上分出一定的磁盘空间，用户可以租用此部分空间，以供用户放置站点及应用组件，提供必要的数据存放和传输功能。

基于域名的虚拟主机

1) 在 conf 同级目录下新建 test 目录，并在里面新建 index.html 文件

<html>
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
    </head>
    <body>
        <h2>基于域名访问-index</h2>
    </body>
</html>

2) 进入 conf 目录编辑 nginx.conf 文件
3) 定位至 http 节点

http {
    include       mime.types;
    default_type  application/octet-stream;  
    sendfile        on;  
    keepalive_timeout  65;        
    server {
        listen       80;
        server_name  localhost;
        location / {
            root   html;
            index  index.html index.htm;            
        }
    }
}

4) 新增加一个 server 节点

server {
        listen       80;
        server_name  marklogzhu;
        charset utf-8;
        location / {
            root   test;           
        }
}

5) 配置 windows 本机 host 文件(注：否则无法通过域名访问)

打开C:\Windows\System32\drivers\etc下面的hosts文件 新增记录
127.0.0.1        marklogzhu

6) 重启 Nginx 

nginx -s reload

7) 浏览器中输入 http://marklogzhu

基于端口的虚拟主机

1) 进入 conf 目录编辑 nginx.conf 文件
2) 定位至 http 节点
3) 新增加一个 server 节点

server {
        listen       8888;
        server_name  marklogzhu;
        charset utf-8;
        location / {
            root   test;           
        }
}

4) 重启 Nginx 并在浏览器中输入 http://marklogzhu:8888
