1.什么是 HTML

HTML 是英文 Hyper Text Mark-up Language( 超文本标记语言 )的缩写，它规定了自己的语法规则，用来表示比“文本”更丰富的意义，比如图片，表格，链接等。

简单来说 HTML 就是给文本加上各种表明文本含义的标签，让人和程序对文本有更直观的认识。

form表单用 GET 方法进行 URL 传值时后台无法获取问题

• 2017年08月07日

问题描述

<a href="${pageContext.request.contextPath}/client?method=add">点我</a>

<form action="${pageContext.request.contextPath }/client?method=add" method="get">
    <p>First name: <input type="text" name="fname" /></p>
    <p>Last name: <input type="text" name="lname" /></p>
    <input type="submit" value="Submit" />
</form>

form 表单使用 GET 方式提交数据，后台打印不出来参数 method 的值，在浏览器地址栏中也不显示值，地址栏显示如下：

http://localhost:8888/CRM/client?fname=1&lname=1

使用 POST 方式地址栏就能显示参数 method

http://localhost:8888/CRM/client?method=addUser

什么是 hexo

hexo 是博客框架，将支持的类型的文件转换成静态Web页面（html+css+JavaScript）。

常用命令

  • hexo n “我的博客” == hexo new “我的博客” #新建文章
  • hexo p == hexo publish
  • hexo g == hexo generate :生成
  • hexo s == hexo server :启动服务预览
  • hexo s -p 5000 == hexo server :启动服务修改端口为5000预览
  • hexo d == hexo deploy :部署

一、JSP介绍

什么是 JSP？

JSP全称是 Java Server Pages，它和 servle 技术一样，都是 SUN 公司定义的一种用于开发动态 web 资源的技术。

JSP 用来干什么？

相对于 HTML 只能为用户提供静态数据，Jsp 技术允许在页面中嵌套 java 代码，为用户提供动态数据。实现将内容的生成和信息的展示相分离。
