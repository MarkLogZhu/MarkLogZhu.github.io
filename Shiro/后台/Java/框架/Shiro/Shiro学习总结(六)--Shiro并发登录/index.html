Shiro学习总结(六)--Shiro并发登录

Shiro Java Shiro 2018/05/23

什么是并发登录

并发登录指的是同一个账号在多个地方同时登录，有些项目允许同时同类，有些项目为了安全考虑只允许一个人登录，这个时候要么不让后者登录；要么踢出前者登录（强制退出）。Shiro 没有提供默认实现，不过可以很容易的在 Shiro 中加入这个功能。通过 Shiro Filter 机制扩 KickoutSessionControlFilter 完成。

核心依赖

在 com.marklogzhu.ssm.shiro新建 KickoutSessionControlFilter.java 文件

在 resources/ehcache 目录下新建 ehcache.xml 文件

在 resources/spring 目录下编辑 applicationContext-shiro.xml

启动项目并运行
