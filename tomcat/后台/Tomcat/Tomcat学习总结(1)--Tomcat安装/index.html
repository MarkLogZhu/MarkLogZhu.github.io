Tomcat学习总结(1)--Tomcat安装

tomcat tomcat 2018/04/11

什么是tomcat

Tomcat 是一个免费的开放源代码的 Web 应用服务器，属于轻量级应用服务器，在中小型系统和并发访问用户不是很多的场合下被普遍使用，是开发和调试 JSP 程序的首选。

注：在安装 tomcat 前电脑必须有 jdk 环境，否则将无法启动 tomcat

安装

1) 前往 官网 下载指定版本

2) 下载系统对应版本的 zip 文件并解压到指定目录下

目录结构

  • bin : 可执行程序,如启动服务、关闭服务
  • conf: 配置文件，可以配置端口、内存大小等
  • lib: tomcat 运行时依赖的 jar 包
  • logs: tomcat 运行时的日志文件
  • temp: tomcat 临时文件
  • webapps: 存放 web 项目的目录，其中每个文件夹都是一个项目
  • work: 运行时生成的文件，最终运行的文件都在这里。通过webapps中的项目生成的
  • license:许可证
  • notice: 说明文件

运行

1) 进入bin 目录 ，双击运行 startup.bat

2) 打开浏览器输入 http://localhost:8080/

出现上图所示，就说明tomcat安装成功了

将tomcat窗口关闭并刷新浏览器，发现无法访问默认的页面了。

这是因为我们将tomcat服务关闭了自然就不能打开页面了，那如何可以不显示tomcat窗口也能正常使用tomcat服务呢？
有两种方案：

* 使用安装版tomcat
* 解压版tomcat注册服务

注册服务

1) 运行cmd 程序进入 tomcat 下的 bin 目录
2) 执行安装服务命令 service.bat install

注：如果 bin 目录下没有 service.bat 文件，请去官网下载有该文件的版本

卸载服务

1) 运行cmd 程序进入 tomcat 下的 bin 目录
2) 执行安装服务命令 service.bat remove
