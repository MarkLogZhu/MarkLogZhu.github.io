MySQL学习总结(六)--函数

MySQL MySQL 2018/06/14

函数

函数表示对输入参数返回一个具有特定关系的值。MySQL 提供了大量函数，从功能区分有如下几类：

  • 数学函数
  • 字符串函数
  • 日期和时间函数
  • 条件判断函数
  • 系统函数
  • 加解密函数

数学函数

数学函数用来处理数值数据，在函数运算过程中有错误发生，函数返回 null 值，主要有：

  • 绝对值函数
  • 三角函数
  • 对数函数
  • 随机函数
函数名 作用 实例 结果
ABS(X) 返回绝对值 ABS(-3) 3
PI() 返回圆周率 PI() 3.141593
SQRT(X) 返回平方根 SQRT(9) 3
MOD(X,Y) 返回求余结果 MOD(31,8) 7
CEIL(X) 返回不小于X的最小整数值 CEIL(-3.5) -3
FLOOR(X) 返回不大于X的最大整数值 FLOOR(-3.5) -4
… … … …

字符串函数

字符串函数用来处理字符串数据,主要有:

  • 字符串长度函数
  • 字符串合并函数
  • 字符串替换函数
  • 字符串查找函数
  • 字符串比较函数
函数名 作用 实例 结果
CHAR_LENGTH(str) 返回字符串 str 的字符个数 CHAR_LENGTH(‘ABCD’) 4
LENGTH(str) 返回字符串 str 的字节长度(一个汉字占三字节长度) LENGTH(‘ACD’) 4
CONCAT(s1,s2,…) 将字符串连接，如果其中一个参数为 null，结果返回 null；如果其中一个参数是二进制字符串，结果返回二进制字符串 CONCAT(‘A’,’B’) AB
CONCAT_WS(x,s1,s2,…) 将字符串以X代表的分隔符相连 CONCAT_WS(‘-‘,’A’,’B’) A-B
INSERT(s1,x,len,s2) 将 S1 字符串从第 X 位开始，替换为 S2 的 len 长度 INSERT(‘ABCDEFG’,2,3,’1234’) ‘A123EFG’
LOWER(sre) 将 str 字符串全部转为小写 LOWER(‘ABCD’) abcd
LCASE(sre) 将 str 字符串全部转为小写 LCASE(‘ABCD’) abcd
UPPER(sre) 将 str 字符串全部转为大写 UPPER(‘abcd’) ABCD
UCASE(sre) 将 str 字符串全部转为大写 UCASE(‘abcd’) ABCD
LEFT(s,n) 获取字符串 s 最左边的 n 个字符 LEFT(‘ABCDE’,2) ab
RIGHT(s,n) 获取字符串 s 最右边的 n 个字符 RIGHT(‘ABCDE’,2) de
LPAD(s1,len,s2) 将S2填充到S1的左边，如果len的长度s1小，那么将返回s1的长度缩到 len SELECT LPAD(‘ABCDEFG’,1,’1234’) A
RPAD(s1,len,s2) 将S2填充到S1的右边，如果len的长度s1小，那么将返回s1的长度缩到 len RPAD(‘ABCDEFG’,1,’1234’) A
LTRIM(s) 删除字符串左边的空格 LTRIM(‘ AB’) AB
RTRIM(s) 删除字符串右边的空格 RTRIM(‘AB ‘) AB
TRIM(s) 删除字符串两侧的空格 TRIM(‘ AB ‘) AB
TRIM(s1 FROM s) 删除s字符串两侧 s1字符 TRIM(‘A’ FROM ‘ABABA’) BAB
REPEAT(s,n) 返回一个由重复字符串 s 组成的字符串，n 表示重复次数 REPEAT(‘s22’,2) s22s22
SPACE(n) 返回一个由 n 个空格组成的字符串 SPACE(2)

REPLACE(s,s1,s2) 使用字符串s2替代字符串s中所有的字符串s1 REPLACE(‘xxx.marklogzhu.com’,’x’,’www’) www.marklogzhu.com
STRCMP(s1,s2) 比较两个字符串大小，s1>s1 返回 -1 否则返回 1 STRCMP(‘A’,’B’) -1
SUBSTRING(s,n,len) 从字符串s的n位置返回一个len长度的字符串(如果 n 为负数,起始点位于字符串倒数第N个字符) SUBSTRING(‘ABCDEFG’,-2,5) FG
MID(s,n,len) 从字符串s的n位置返回一个len长度的字符串(如果 n 为负数,起始点位于字符串倒数第N个字符) MID(‘ABCDEFG’,-2,5) FG
LOCATE(str1,str) 返回子字符串 s1 在字符串 str 中开始的位置 LOCATE(‘D’,’ABCDEFG’) 4
POSITION(str1,str) 返回子字符串 s1 在字符串 str 中开始的位置 POSITION(‘D’,’ABCDEFG’) 4
INSTR(str1,str) 返回子字符串 s1 在字符串 str 中开始的位置 INSTR(‘D’,’ABCDEFG’) 4
REVERSE(s) 将字符串 s 反序输出 REVERSE(‘ABCDEFG’) GFEDCBA
ELT(N,字符串1,字符串2,字符串3,…) 返回n位置的字符串，如果N>字符串个数则返回 NULL ELT(2,’A’,’B’,’C’) B
FIELD(s,s1,s2,…) 返回字符串s在列表中第一次出现的位置，如都不存在返回0 FIELD(‘A’,’AB’,’B’,’A’) 3
FIND_IN_SET() 返回字符串 s1 在字符串列表 s2 中出现的位置，字符串列表以逗号间隔，不不存在返回0 ，任意一参数为 null 则返回 null;第一个函数含逗号则返回0 SELECT FIND_IN_SET(‘A,’,’BCD,BBB,ABC,A,’) 0
MAKE_SET(x,s1,s2,…) 返回由 x 的二进制指定的相应位的字符串组成的字符串 MAKE_SET(1 4,’A’,’B’,’C’,’D’) A,C

日期和时间函数

日期和时间函数主要用来处理日期和时间。

函数名 作用 实例 结果
CURDATE() 返回当前系统日期 CURDATE() 2018-06-14
CURRENT_DATE() 返回当前系统日期 CURRENT_DATE() 2018-06-14
CURRENT_TIMESTAMP() 返回当前系统日期和时间 CURRENT_TIMESTAMP() 2018-06-14 14:30:20
NOW() 返回当前系统日期和时间 NOW() 2018-06-14 14:30:20
SYSDATE() 返回当前系统日期和时间 SYSDATE() 2018-06-14 14:30:20
UNIX_TIMESTAMP(data) 返回 Unix 时间戳 UNIX_TIMESTAMP(NOW()) 1528957984
UTC_DATE() 返回当前UTC(世界标准时间)日期值 UTC_DATE() 2018-06-14
UTC_TIME() 返回当前UTC(世界标准时间)时间值 UTC_TIME() 06:37:38
MONTH(date) 获取月份 MONTH(NOW()) 6
MONTHNAME(date) 获取月份对应的英文名 MONTH(NOW()) June
DAYNAME(date) 获取日期对应的工作日英文名称 DAYNAME(NOW()) Thursday
DAYOFWEEK(date) 获取日期对应的一周的位置.1表示周日 DAYOFWEEK(NOW()) 5
WEEKDAY(date) 获取日期对应的一周的位置.0表示周一 WEEKDAY(NOW()) 3
WEEK(date) 返回日期对应的周 WEEK(NOW()) 23
DAYOFYEAR(data) 返回日期对应的一年中的第几天 DAYOFYEAR(NOW()) 165
DAYOFMONTH(data) 返回日期对应的一月中的第几天 DAYOFMONTH(NOW()) 14
YEAR(date) 返回日期对应的年份 YEAR(NOW()) 2018
QUARTER(date) 返回日期对应的季度 QUARTER(NOW()) 2
MINUTE(date) 返回日期对应的分钟值 MINUTE(NOW()) 28
SECOND(date) 返回日期对应的秒数 SECOND(NOW()) 51
… … … …

条件判断函数

条件判断函数也叫流程控制函数，根据条件不同执行相应的流程。

函数名 作用 实例 结果
IF(expr,v1,v2) 表达式 expr 成立返回 v1 否则返回 v2 SELECT IF(1>0,1,2) 1
IFNULL(v1,v2) 如果V1不为NULL，返回V1,否则返回V2 SELECT IFNULL(null,2) 2
CASE expr WHEN v1 THEN r1 ELSE r2 END 判断表达式值是否是V1 ，如果是返回R1，或者返回r2 CASE 1 WHEN 1 THEN 1 ELSE 2 END 1

系统函数

系统函数用于获取 MySQL 的信息。

函数名 作用 实例 结果
VERSION() 获取 MySQL 版本号 VERSION() 5.7.20-log
CONNECTION_ID 返回 MySQL 服务器当前连接次数 CONNECTION_ID() 3
SHOW PROCESSLIST; 显示当前用户的连接信息 SHOW PROCESSLIST;
USER() 获取用户名 USER() root@localhost
CURRENT_USER() 返回用户名 CURRENT_USER() root@localhost
SESSION_USER() 返回用户名 SESSION_USER() root@localhost
CHARSET(str) 返回字符串 str 的字符集格式 CHARSET(‘ABC’) utf8mb4
COLLATION(str) 返回字符串 str 的排列方式 COLLATION(‘ABC’) utf8mb4_general_ci
LAST_INSERT_ID() 返回最后一个插入或更新主键的值 LAST_INSERT_ID() 1

加/解密函数

加解密函数主要用来对数据进行加解密操作，以保证数据不被他人获取。

函数名 作用 实例 结果
PASSWORD(str) 用于加密后mysql用户密码，参数为 nul 时返回 null PASSWORD(‘marklogzhu@163.com’) *DF9D970C6D333136D336C6BE1ECFAD031C8ABF08
MD5(str) 以MD5形式加密字符串 MD5(‘123’) 202cb962ac59075b964b07152d234b70
