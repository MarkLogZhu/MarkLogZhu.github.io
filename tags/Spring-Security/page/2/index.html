Spring Security学习总结(二)--基于表单的认证

• 2019年05月01日

原理

我们先来看下表单认证的过滤链：

  • UsernamePasswordAuthenticationFilter: 基于用户名和密码的表单登陆认证过滤器
  • BasicAuthenticationFilter: 基于 httpBasic 登陆认证过滤器
  • ExceptionTranslationFilter: 捕获 Security 过滤器中出现的异常
  • FilterSecurityInterceptor: Security 过滤器最后一环，根据配置的规则检验是否可以访问 REST API
  • REST API: web 资源，如：controller 接口方法

注：绿色的过滤器根据实际需要进行配置，其他的过滤器必须存在。
