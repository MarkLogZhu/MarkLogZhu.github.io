Nginx学习总结(3)--静态文件服务器

Nginx Nginx 2018/04/13

什么是静态资源

非服务器动态运行生成的文件，如：

  • html,css,js
  • 图片
  • 视频
  • 文件

显示图片

1) 进入 conf 目录编辑 nginx.conf 文件
2) 在 server 节点下新增加一个 location 节点

location ~ .*\.(gif|jpg|jpeg|png)$ {
        root   D:\images;#图片存放目录                
}

3) 重启 Nginx 

nginx -s reload

4) 浏览器中输入 http://localhost/test.jpg
