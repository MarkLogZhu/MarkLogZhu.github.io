Swagger2 是什么？

Swagger2 是一款RESTFUL接口的文档在线自动生成+功能测试功能软件。

Swagger2 是一个规范和完整的框架，用于生成、描述、调用和可视化 RESTful 风格的 Web 服务。总体目标是使客户端和文件系统作为服务器以同样的速度来更新。文件的方法，参数和模型紧密集成到服务器端的代码，允许 API 来始终保持同步。Swagger2 让部署管理和使用功能强大的 API 从未如此简单。

SpringBoot学习总结(三)--集成 SpringDataJpa

• 2018年07月23日

在上一章节中，我们讲到了除了 Mybatis 外，还有另一种 orm 框架，今天我们就来学习下怎么集成使用。

什么是 spring data jpa

Spring Data JPA 是 Spring 基于 ORM 框架 和 JPA 规范的基础上封装的一套 JPA 应用框架，可使开发者用极简的代码即可实现对数据的访问和操作。它提供了包括增删改查等在内的常用功能，且易于扩展！学习并使用 Spring Data JPA 可以极大提高开发效率！

spring data jpa 让我们解脱了 DAO 层的操作，基本上所有 CRUD 都可以依赖于它来实现。

SpringBoot学习总结(一)--Hello World

• 2018年07月17日

如果有使用过 SSM 框架经验的同学肯定知道，在运行一个项目前你需要配置相当多的配置。复杂繁琐的配置让新手无所适从，而此时【此处应有掌声】SprignBoot 闪亮登场。

什么是 SpringBoot

Spring Boot 是由 Pivotal 团队提供的全新框架，其设计目的是用来简化 Spring 应用的初始搭建以及开发过程。该框架使用了特定的方式来进行配置，从而使开发人员不再需要定义样板化的配置。简单来说，就是实现 约定大于配置 这一口号，你通过默认的设置就能正常启动一个项目啦！
