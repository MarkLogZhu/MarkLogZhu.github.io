2018/03/09

1.点击

2.对话框中依次点击

2.1) 选中 Java Enterprise 选项卡
2.2) 勾选 Web Application(3.1) 复选框，这表示我们新建的项目是个网站。
2.3)    点击 Next 按钮，这会在 project下添加一个默认的与 project 同名的 module。
  1. 输入项目名称和存放位置，然后点击 finish 按钮确认

4.在 WEB-INI 目录下新建 lib 和 classes 包，用来存放 jar 包跟编译后的类文件

4.1) 在 WEB-INI 目录上右键
4.2)    选择 NEW 
4.3)    选择 Directory
4.4)    取名为 lib/classes

5.配置文件夹路径

5.1) 打开 Project Structure 对话框

5.2) 选择 Module  --> Paths --> 选择 Use module compile output path  -> 将O utput path 和 Test output path都选择刚刚创建的 classes 文件夹。

5.3) 选择 Dependencies --> 点击右边的 + 号  --> 选择 Jars or Directories --> 选择 jar Directory

6.配置Tomcat

6.1) 打开菜单 Run --> 选择 Edit Configuration 

6.2) 点击 + 号  --> 拉倒最后，选择 9 items mort...  --> 选择 Tomcat Server --> Local

6.3) 设置 Tomcat 的名称和安装路径（注：下面的 Error 提示是因为还没有部署项目）

7.部署项目并运行

7.1) 选择Deployment --> 点击 + 号  -->  选择 Artifact --> 点击 OK 按钮

7.2)在 index.jsp 文件中输入 <h1>Hello world</h1> 并保存

7.3) 点击 Tomcat 启动

7.4) 在弹出的浏览器中就会显示刚刚我们敲入的 Hello world 了     

(注：如果你在配置 Tomcat 时将afer launrch 勾去掉的话，就需要自己启动浏览器并输入地址了)
