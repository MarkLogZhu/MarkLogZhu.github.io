Velocity实战入门(三)--赋值

Velocity Velocity 2018/04/25

在上一篇中我们已经了解了怎么取值，现在就让我们来学习下怎么赋值

赋值方式

赋值方式有两种：

* 后台赋值
* 页面 #set 指令赋值

后台赋值

我们先来看看后台是怎么赋值的

@RequestMapping("/index2")
public String vmIndex2(Model model) {
    model.addAttribute("user", initUser());
    model.addAttribute("args",initArgs());
    return "test2";
}

private User initUser(){
    User user = new User();
    user.setUserName("marklogzhu");
    user.setAge(18);
    return user;
}

private String[] initArgs(){
    return new String[]{"张三","李四","王五","赵六"};
}

可以看到是直接声明一个对象并把它添加到 model 对象中去，最后由 model 放置到 request 作用域里

页面 #set 指令赋值

除了在后台赋值的方式外还可以直接在页面通过 #set 指令来赋值

1) 在 HelloController.java 文件中增加如下方法：

@RequestMapping("/index3")
public String vmIndex3(Model model) {
    model.addAttribute("age","5");
    return "test3";
}

2) 在 templates 目录下新建 test3.vm 文件

#set ($name = "marklogzhu")
#set ($age = 18)
#set ($class = $ccc)
#set($args = ["张三", "李四", "王五", "赵六"])
<html>
<head>
    <title>Spring MVC and Velocity</title>
</head>
<body>
    页面赋值对象:$name
    <br>
    后端赋值为5，页面赋值为18，结果为：$age
    <br>
    #set ($age = $age+1)
    <br>
    通过\#set指令加 1 后的值： $age
    <br>
    赋值对象未定义情况结果为: $class
</body>
</html>

3) 启动项目并在浏览器中输入 http://localhost:8080/hello/index3

从上面的结果我们可以发现：

* #set($key = value) 的形式可以在页面赋值
* 后端跟页面对同一个变量赋值时，页面会覆盖掉后端的值
* #set 赋值时 value为 null 时，他的值就是 kye 的字面量
* 没有 $age++、++$age 这种自增方式
