SpringMVC之JSON返回日期格式化

SpringMVC Java SpringMVC 2018/05/30

在使用 SpringMVC 框架返回 JSON 数据时，默认返回的日期型是数值格式。那么可以返回 yyyy-MM-dd HH:mm:ss格式吗，今天我们就要实现。

在 springmvc.xml 文件中配置

关键配置

配置前返回数据格式

{
    "status": 0,
    "msg": "登录成功",
    "data": {
        "userId": 10000000,
        "userName": "admin",
        "telephone": "135XX72XXXX",
        "mail": "marklogzhu@163.com",
        "password": "",
        "status": 1,
        "remark": "",
        "operator": "admin",
        "operateTime": 1527575674000,
        "operateIp": "127.0.0.1"
    },
    "success": true
}

配置后返回数据格式

{
    "status": 0,
    "msg": "登录成功",
    "data": {
        "userId": 10000000,
        "userName": "admin",
        "telephone": "135XX72XXXX",
        "mail": "marklogzhu@163.com",
        "password": "",
        "status": 1,
        "remark": "",
        "operator": "admin",
        "operateTime": "2018-05-29 14:34:34",
        "operateIp": "127.0.0.1"
    },
    "success": true
}

可以看到 operateTime 字段格式已经变为 yyyy-MM-dd HH:mm:ss 格式。
