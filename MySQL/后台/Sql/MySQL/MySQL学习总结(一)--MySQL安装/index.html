MySQL学习总结(一)--MySQL安装

MySQL MySQL 2018/06/12

MySQL 简介

MySQL 是一个关系型数据库管理系统，由瑞典 MySQL AB 公司开发，目前属于 Oracle 公司。MySQL 是一种关联数据库管理系统，关联数据库将数据保存在不同的表中，而不是将所有数据放在一个大仓库内，这样就增加了速度并提高了灵活性。

  • MySQL 是开源的，所以你不需要支付额外的费用。
  • MySQL 支持大型的数据库。可以处理拥有上千万条记录的大型数据库。
  • MySQL 使用标准的SQL数据语言形式。
  • MySQL 可以运行于多个系统上，并且支持多种语言。这些编程语言包括C、C++、Python、Java、Perl、PHP、Eiffel、Ruby和Tcl等。
  • MySQL 对PHP有很好的支持，PHP是目前最流行的Web开发语言。
  • MySQL 支持大型数据库，支持5000万条记录的数据仓库，32位系统表文件最大可支持4GB，64位系统支持最大的表文件为8TB。
  • MySQL 是可以定制的，采用了GPL协议，你可以修改源码来开发自己的 MySQL 系统。

Windows 环境下安装

1) 前往 官网 下载指定版本安装包

2) 下载完毕后解压，一直点击下一步安装即可

3) 以管理员身份运行 CMD，并进入 MySQL 安装目录

4) 输入 net strat mysql 启动服务（net stop mysql 为关闭服务）

5) 安装/移除服务

[服务名]为可选参数，默认不填写则为 MySQL

Linux 环境下安装

在 CentOS 中默认安装有 MariaDB，这个是 MySQL的分支，但为了需要还是要在系统中安装 MySQL ，而且安装完成之后可以直接覆盖掉 MariaDB。

1）查询是否安装 mysql

rpm -qa | grep mysql

2) 下载并安装MySQL官方的 Yum Repository (这个文件服务器无法下载，我是本地下载完上传到服务器的)

wget -i -c http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql57-community-release-el7-10.noarch.rpm

3）安装 mysql 

yum -y install mysql-community-server

4）启动服务

-- 查看服务状态
    systemctl stauts mysqld.service
-- 启动服务
    systemctl start mysqld.service
-- 关闭服务
    systemctl stop mysqld.service

5) 修改密码

-- 登陆mysql (第一次登陆无需密码，后面登陆都需要) 
    mysql -uroot
-- 切换至 mysql 数据库
    use mysql;
-- 修改密码
    update user set password = password('新密码') where user = 'root'
-- 配置刷新
    flush privileges;

6) 开启远程访问

-- 卸载密码验证插件 validate_password
    UNINSTALL PLUGIN validate_password ;
-- 开启 mysql 远程访问
    grant all privileges on *.* to 'root' @'%' identified by 'sa000'
-- 配置刷新
    flush privileges;

7) 开启防火墙端口

-- 放行3306端口
    firewall-cmd --zone=public --add-port=3306/tcp --permanent
-- 重启防火墙
   firewall-cmd --reload

8) mysql 服务开机自启动

-- 进入启动项文件
    vim /etc/rc.local
-- 在最后面添加    
    service mysqld start
-- 重启系统
    shutdown -r now
-- 测试 mysql 是否启动    
    service mysql status

登录 MySQL

1）输入 mysqladmin -u root password 密码 设置密码（初次登陆没有密码需要先设置）

2）输入 mysql -u root -p 回车，然后输入密码
