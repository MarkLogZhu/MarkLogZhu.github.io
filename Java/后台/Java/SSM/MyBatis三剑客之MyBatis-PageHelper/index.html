MyBatis三剑客之MyBatis-PageHelper

Java Java MyBatis SSM 2018/05/11

mybatis 有很多优点，但是也有很多缺点，比如今天我们要讲的分页功能，在 mybatis 中要使用分页功能需要如下操作：

* 查询分页数据
* 获取分页长度

也就是说需要调用两个方法才能完成分页功能，这就显得很繁琐。幸好这个问题不止我遇到，早有大佬开发了一个分页插件来简化我们的开发,那就是 pagehelper分页插件。它的原理是利用 mybatis拦截器，在查询数据库的时候，拦截下 SQL，然后进行修改，从而实现分页功能。
