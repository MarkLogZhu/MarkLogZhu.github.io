RabbitMQ学习总结(八)--整合SpringBoot

• 2018年01月02日

之前我们学会了如何使用 Spring AMQP ，今天我们来整合 SpringBoot 使用。

SpringBoot 是在 Spring AMQP 上面封装了一层，不需要再像 Spring AMQP 一样注入各个组件 Bean, 只需要在配置文件上配置好 RabbitMQ 属性，SpringBoot 就可以自动注入了。

使用 @RabbitListener 注解可以实现消费端的处理。

概念讲解

注解

@RabbitListener 注解指定目标方法来作为消费消息的方法，通过注解参数指定所监听的队列或者Binding。使用 @RabbitListener 可以设置一个自己明确默认值的 RabbitListenerContainerFactory 对象。
