Beetl Java Beetl 模板引擎 2018/08/25

Beetl 资料太少，官网的文档某些方面不太适合像我这种小白入门，所以有了这个系列的博客。内容基本上都是拷贝官网的，但会从一个个实例来演示如何使用。

Beetl 是什么

简单来说就是一个用 Java 写的一个模板引擎。

Beetl官网

接触一门技术，不要管它的原理是什么，先要能在本地跑起来，我们就先来实现下环境搭建。
