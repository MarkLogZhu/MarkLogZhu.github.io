基于 XML 的自动装配

除了上面手动配置 XML 的形式注入外，Spring还为我们提供了自动向 Bean 注入依赖的功能，这个过程一般被称为自动装配（autowiring)。

Spring 的自动装配有三种模式：byTpye(根据类型)，byName(根据名称)、constructor(根据构造函数)。

byTpye 模式

在 byTpye 模式中，Spring 容器会基于反射查看 bean 定义的类，然后找到与依赖类型相同的 bean 注入到另外的 bean中，这个过程需要借助 setter 注入来完成，因此必须存在 set 方法，否则注入失败。
